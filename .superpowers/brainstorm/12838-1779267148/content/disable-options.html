Wie sollen deaktivierte Optionen aussehen?

Wenn ein Agent versucht, eine Ressource einzureichen, die beauftragt oder nicht verfügbar ist

Option A: Vollständig versteckt

Kontextmenü der Ressource:

🖊️ Bearbeiten
🗑️ CV löschen
🚫 Deaktivieren

→ "Profil einreichen" ist nicht im Menü

KI-Match Panel:

Vakanz wählen...
⚠️ Nicht verfügbar für Einreichung

A: Verstecken

Option verschwindet komplett → sauberer, aber User könnte verwirrt sein

Option B: Ausgegraut + Tooltip

Kontextmenü der Ressource:

🖊️ Bearbeiten
🗑️ CV löschen
📤 Profil einreichen
🚫 Deaktivieren

Hover zeigt: "Ressource ist beauftragt bis 31.05.2026"

KI-Match Panel:

Vakanz wählen...
⚠️ Ressource ist beauftragt bis 31.05.2026

B: Ausgegraut + Grund anzeigen

Option sichtbar aber deaktiviert, mit Tooltip warum → mehr Transparenz
